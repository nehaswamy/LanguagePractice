# Language Practice 
Language Practice is an application that allows users to practice vocabulary words of a language they are studying according 
to difficulty. Specifically, if a user fails to match the vocabulary word with its definition, the word will be added to a "words to study"
list. This way, a user can focus on words that are more difficult but can keep all vocabulary words in one place. 

## How to Play
* Clone this repository (git clone https://github.com/nehaswamy/LanguagePractice) 
* Run src/GUI.java
